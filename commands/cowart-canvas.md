# Cowart Canvas

Open the Cowart canvas for this project.

## Workflow

1. Start the Cowart local service from `/Users/anita/plugins/cowart` if it is not already running.
2. Use the active Codex project directory as the Cowart project directory. Do not use the Cowart plugin repository as the project directory.
3. Open `http://127.0.0.1:43217/` in the Codex in-app browser when browser control is available.
4. If browser control is unavailable, report the local URL and keep the Cowart service running.

## Output

Reply in Simplified Chinese with the canvas URL and the project-local canvas data directory.
